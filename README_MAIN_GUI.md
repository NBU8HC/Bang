## 🔍 Troubleshooting

### "Cannot find tool" error
- **Cause:** First extraction failed
- **Fix:** Delete main_gui.exe, re-download, run as Admin

### Slow launch
- **Cause:** Antivirus scanning
- **Fix:** Add main_gui.exe to antivirus exclusions

### Tool doesn't launch
- **Cause:** Temp folder full or write-protected
- **Fix:** Check %TEMP% folder, run as Admin
